# 🎧 NexGen Connect

**Sistema Dinâmico para Gestão de Chamados de Suporte em Pequenas Empresas**

## 📌 Sobre o Projeto
O NexGen Connect é uma aplicação web responsiva (Single Page Application) desenvolvida para centralizar, gerenciar e monitorar o ciclo de vida de chamados técnicos. O projeto resolve o problema da ineficiência e perda de histórico gerados pelo uso de canais informais de comunicação em pequenas empresas.

## 🚀 Tecnologias Utilizadas
* **React:** Biblioteca JavaScript para construção da interface de usuário modular.
* **Vite:** Ferramenta de build e servidor de desenvolvimento ultrarrápido.
* **Tailwind CSS:** Framework CSS utilitário para a estilização responsiva e padronização visual.
* **HTML5 Semântico:** Estruturação focada em boas práticas e acessibilidade.

## ⚙️ Como Instalar e Rodar Localmente

Siga os passos abaixo para executar a aplicação no seu ambiente local:

**1. Clone este repositório:**
```bash
git clone https://github.com/IoT-Luiz-Filipe99/nexgen-connect-frontend.git
2. Acesse a pasta do projeto:

Bash
cd nexgen-connect-frontend
3. Instale as dependências necessárias:

Bash
npm install
4. Inicie o servidor de desenvolvimento:

Bash
npm run dev
5. Abra o navegador e acesse: http://localhost:5173

👥 Funcionalidades do MVP (Produto Mínimo Viável)
Portal do Colaborador: Interface otimizada para o usuário final relatar incidentes de forma clara, definindo título, nível de prioridade (Baixa, Média, Alta) e descrição.

Painel Técnico (Dashboard): Visão administrativa para a equipe de TI visualizar a fila de chamados e gerenciar o status de cada ticket (Aberto, Em Andamento, Concluído).
